# LG UX Font Generator

Python script to generate font apks from font files for lg ux. 

## Requirements

* Java >=8 
* Python >=3.6

## Instructions

1. ```pip install -r requirements.txt```
2. ```python main.py -h```

## Misc

* Source code for the decomplied font apk: https://github.com/ssnjr2002/LGFontApk
